# Photographer Studio (Sanity)

This is the Sanity Studio for managing your content.

## Configure

Create a Sanity project and set these env vars:

- `SANITY_STUDIO_PROJECT_ID`
- `SANITY_STUDIO_DATASET`
- `SANITY_STUDIO_API_VERSION` (example: `2024-01-01`)

## Run

From the repo root:

- `npm install`
- `npm run dev:studio`
